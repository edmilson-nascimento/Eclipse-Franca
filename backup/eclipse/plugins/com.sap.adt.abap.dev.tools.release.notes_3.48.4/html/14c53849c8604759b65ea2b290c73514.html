ABAP Development Tools for Eclipse

ABAP Memory Snapshot

The ABAP Memory Snapshot view lists all memory snapshots. From here, you can open an ABAP memory snapshot in the editor.

In the editor, the Memory Objects tab displays a list with the memory objects ranked by their size.

Sample of memory objects, ranked in accordance with the value Bound (Allocated)

Here, the following information is displayed:

  1. Integrated toolbar

  2. List of memory objects

  3. Memory sizes

These views enable you to analyze the memory usage of an ABAP application.

For more information, see ABAP Memory Snapshots.

ABAP Cross Trace: Exporting and Importing a Trace Result

You can now export/import a trace result of the ABAP Cross Trace to/from a JSON file. This enables you to store important trace results on a file system and to collaborate with other colleagues on error analysis.

For more information, see

  • Exporting an ABAP Cross Trace Result and

  • Importing an ABAP Cross Trace Result.

Develop Custom Actions with the IDE Action Framework and Enhance Them with Server-Driven UI

The IDE Action framework allows you to extend the ABAP development tools for Eclipse with custom actions. You can create and run IDE Actions, which are additional functionalities implemented completely in the backend. On top of that, you have the option to use server-driven UI APIs to build UIs, add parameters and additional configurations to your actions, such as side effects and custom value helps.

For more information, see Working with IDE Actions and Working with Server-Driven UIs.

CDS External Entity in Data Preview

The CDS external entity is now supported in Data Preview.

For more information, see Configuring Select Options for Data Preview.

AMDP Test Double Framework

You can now write automated tests for your ABAP Managed Database Procedure (AMDP) using the new AMDP test double framework. AMDP test double framework helps in testing the logic written in an AMDP while writing the ABAP Unit tests. This API-based framework can be used to isolate and test the logic encapsulated within an AMDP method by isolating your AMDP from all the actual dependencies associated with the code being tested and replacing these dependencies with test doubles created for them.

For more information, see AMDP Test Double Framework.

CDS Test Double Framework: Supported Test Scenario

CDS test double frameworks now supports testing of CDS Hierarchy view.

For more information, see ABAP CDS Test Double Framework.

Displaying the Original WSDL in a SOAP Provider Model

You can now display the original WSDL file that you used to create a SOAP provider model.

For more information, see Creating SOAP Service Providers.

Working with Intelligent Scenarios and Intelligent Scenario Models

You can now work with Intelligent Scenarios and Intelligent Scenario Models in ABAP development tools for Eclipse in addition to the corresponding apps on the Fiori launchpad.

For more information, see Working with Intelligent Scenarios and Working with Intelligent Scenario Models.

 Joule for Developers, ABAP AI Capabilities

Joule for developers, ABAP AI capabilities supports you in the end-to-end development using generative AI in ABAP development tools for Eclipse.

Note
To leverage Joule's capabilities, you need to purchase an additional license:

  • For SAP BTP ABAP environment, see SAP Note 3571857 .

  • In the case of SAP S/4HANA Cloud Public Edition, a dedicated license is required, that is 8019124 - Joule for developers, ABAP AI capabilities - Limited promotion. This dedicated license is a limited offering, which can be acquired until August 15th, 2025 and the usage of the Joule for developers ABAP AI capabilities based on this offering will be available no longer than September 15th, 2025. For standard customers the Account Executive needs to be contacted. Please see KBAs 2626344 and 1660069 for further information.

To use Joule for developers, ABAP AI capabilities, see Prerequisites and Required Authorizations. The following AI-based features are supported:

Joule Chat

The Joule chat supports you during the development process, particularly with ABAP-related questions and tasks. You can converse with the Joule chat in natural language, for example, by asking questions or writing prompts. For more information, see Chat.

Explain

You can get detailed explanations of development objects or selected code using the Explain feature. For more information, see Explain.

Predictive Code Completion

The predictive code completion enables you to use AI-supported code completion in the ABAP source code editor. It allows you to generate ghost texts to predict code, enhancing efficiency in writing code. For more information, see Predictive Code Completion.

ABAP Unit

The AI-based ABAP Unit features are specifically designed to assist you in creating and improving your unit tests more effectively. For more information, see ABAP Unit.

CDS Test Generation

The AI assisted test class generates automated tests for ABAP CDS entity to assist you in generating more context-specific content in the ABAP Unit tests for a CDS entity. For more information, see CDS Test Generation.
