Release Notes of ABAP Development Tools for Eclipse

ABAP development tools for Eclipse is SAP’s state-of-the-art IDE for ABAP development on the open Eclipse platform.

Overview

These release notes are a general overview of the most significant changes relating to ABAP development tools for Eclipse (ADT) features in the context of ABAP development and the ABAP programming models.

Highlighting of SAP Product-Specific Dependencies

ABAP development tools for Eclipse provides features that are client-specific or require a specific back-end version. To highlight and contrast back-end-specific help content in the relevant context, the following SAP icon fonts are used:
Icon Deployment Model Description
 On-premise/Private Cloud Used for SAP NetWeaver, SAP S/4HANA and for SAP S/4HANA Cloud, private edition
 Public Cloud Used for SAP BTP ABAP environment and SAP S/4HANA Cloud

 ADT Feature Availability Matrix for AS ABAP Releases

Most of the ADT features depend on a specific back-end version. To get an overview, which ADT features are enabled with which back-end version, see this blog post.

To see the release notes of the latest ABAP releases for the ABAP Platform that is used for the SAP S/4HANA on premise releases in general, see News for ABAP Release 7.5x

ADT Client Shipment Strategy

ABAP development tools for Eclipse is released to customers in combination with the latest back-end shipments in the context of on-premise/Private Cloud and Public Cloud. Consequently, the documentation of these ADT features will be shipped with the following SAP products:

  •  Documentation in the context of SAP BTP ABAP environment will be shipped in accordance with the relevant SAP BTP shipments.

  •  Documentation in the context of SAP S/4HANA Cloud Public Edition will be shipped in accordance with the relevant SAP S/4HANA Cloud Public Edition shipments.

  •  Documentation in the context of ABAP Platform and SAP S/4HANA Cloud, private edition will be shipped in accordance with the following SAP S/4HANA shipments

For more information about the client shipments from ABAP development tools for Eclipse, see Important changes to the ABAP Development Tools (ADT) release cycle
